# Proyecto Semana 08 — Gestión de Inventario con Arrays

## Dominio

Sistema de Trazabilidad Agrícola

## Objetivo

Implementar un sistema de gestión de inventario utilizando métodos de arrays en JavaScript, aplicados al dominio agrícola.

## Funcionalidades

El programa:

* Define un inventario de cultivos usando un array de objetos
* Usa métodos de mutación (push, pop, unshift, splice)
* Realiza búsquedas con find y filter
* Recorre datos con forEach
* Transforma datos con map
* Utiliza el spread operator para copiar y agregar elementos
* Genera un reporte completo en consola

## Tecnologías utilizadas

* JavaScript (ES6+)
* Node.js

## Ejecución

node starter/script.js
