# Proyecto Semana 06 — Reporte con Bucles

## Dominio
Sistema de Trazabilidad Agrícola

## Objetivo
Procesar un conjunto de datos utilizando diferentes tipos de bucles en JavaScript para generar un reporte completo.

## Funcionalidades

- Listado de cultivos con for...of
- Conteo por categorías
- Cálculo de total y promedio de producción
- Identificación de valores máximo y mínimo
- Generación de reporte con for clásico
- Uso de while para validaciones adicionales
- Uso de break y continue

## Ejecución

node starter/script.js
