# Proyecto Semana 01 - Introducción a la Consola en JavaScript

## Dominio asignado
Sistema de Trazabilidad Agrícola

## Descripción
Este proyecto corresponde a la primera semana del bootcamp.  
El objetivo fue familiarizarse con el uso de **JavaScript y Node.js** para mostrar información en la consola utilizando `console.log()`.

Se creó una salida en consola que representa una **ficha básica del dominio**, en este caso un **Sistema de Trazabilidad Agrícola**, mostrando información relacionada con un lote de cultivo.

## Tecnologías utilizadas
- JavaScript
- Node.js

## Conceptos aplicados

Durante esta actividad se aplicaron los siguientes conceptos:

- Uso de `console.log()` para imprimir información en la consola.
- Organización visual de la salida utilizando separadores y títulos.
- Representación básica de información relacionada con el dominio asignado.
