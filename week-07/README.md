# Proyecto Semana 07 — Librería de Funciones

## Dominio

Sistema de Trazabilidad Agrícola

## Objetivo

Implementar una librería de funciones reutilizables en JavaScript, aplicando buenas prácticas como uso de parámetros, return, funciones puras y separación de lógica.

## Funcionalidades

El programa:

* Define datos de cultivos en un array
* Formatea información usando arrow functions
* Calcula el valor total de un cultivo
* Valida si un cultivo está activo
* Aplica un impuesto usando parámetros por defecto
* Genera un reporte usando funciones como callbacks

## Tecnologías utilizadas

* JavaScript (ES6+)
* Node.js

## Ejecución

node script.js
